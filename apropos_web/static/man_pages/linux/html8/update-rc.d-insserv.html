NAME

update-rc.d-insserv - obsolete

DESCRIPTION

update-rc.d-insserv is an obsolete implementation of the update-rc.d interface for the dependency based boot sequencing system insserv. The implementation was moved into the sysv-rc update-rc.d implementation.
