NAME


select-default-ispell - select default ispell dictionary

SYNOPSIS


select-default-ispell

DESCRIPTION

This program will make debconf always ask the shared question about the default ispell dictionary to be used in your system according to the installed ones, and will do the appropriate settings.
Calls internally update-default-ispell.
