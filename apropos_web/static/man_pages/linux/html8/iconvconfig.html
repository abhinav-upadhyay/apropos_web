NAME

iconvconfig - Create fastloading iconv module configuration file

SYNOPSIS

iconvconfig [OPTION...] [DIR...]

DESCRIPTION

The iconvconfig program generates a cache that internationalised applications can use to reduce loading time.
--prefix PATH
Prefix used for all file accesses
