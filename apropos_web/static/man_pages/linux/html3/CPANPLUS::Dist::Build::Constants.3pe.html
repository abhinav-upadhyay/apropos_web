NAME

CPANPLUS::Dist::Build::Constants - Constants for CPANPLUS::Dist::Build

SYNOPSIS


use CPANPLUS::Dist::Build::Constants;

DESCRIPTION

CPANPLUS::Dist::Build::Constants provides some constants required by CPANPLUS::Dist::Build.
