NAME

asn1_bit_der - API function

SYNOPSIS

#include <libtasn1.h>
 
void asn1_bit_der(const unsigned char * str, int bit_len, unsigned char * der, int * der_len);

ARGUMENTS

const unsigned char * str
BIT string.
int bit_len
number of meaningful bits in STR.
unsigned char * der
string returned.
int * der_len
number of meaningful bytes of DER (der[0]..der[ans_len-1]).

DESCRIPTION

Creates the DER coding for a BIT STRING type (length and pad included).
