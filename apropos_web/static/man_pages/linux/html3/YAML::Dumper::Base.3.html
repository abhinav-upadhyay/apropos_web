NAME

YAML::Dumper::Base - Base class for YAML Dumper classes

SYNOPSIS


package YAML::Dumper::Something;
use YAML::Dumper::Base -base;

DESCRIPTION

YAML::Dumper::Base is a base class for creating YAML dumper classes.
