NAME

MouseX::NativeTraits::CodeRef - Helper trait for CodeRef attributes

SYNOPSIS


package Foo;
use Mouse;

has 'callback' => (
traits => ['Code'],
is => 'ro',
isa => 'CodeRef',
default => sub { sub { print "called" } },
handles => {
call => 'execute',
},
);

my $foo = Foo->new;
$foo->call; # prints "called"

DESCRIPTION

This provides operations on coderef attributes.

PROVIDED METHODS

execute(@args)
Calls the coderef with the given args.
