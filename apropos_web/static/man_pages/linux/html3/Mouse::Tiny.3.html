NAME

Mouse::Tiny - Mouse in a single file

VERSION

This document describes Mouse version 0.97

DESCRIPTION

Mouse::Tiny is Mouse, but it is in a single file.
This is not tiny. In fact, it requires a little more memory and time than Mouse.
Use Mouse unless you know what you are doing.
