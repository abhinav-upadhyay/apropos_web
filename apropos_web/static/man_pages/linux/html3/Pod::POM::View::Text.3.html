Pod::POM::View::Text - create text views of POM objects

DESCRIPTION

Returns the text of a "B<>" sequence in 'bold' (i.e. surrounded by asterisks, like *this*).

Returns the text of a "I<>" sequence in 'italics' (i.e. surrounded by underscores, like _this_).
