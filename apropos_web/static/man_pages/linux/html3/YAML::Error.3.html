NAME

YAML::Error - Error formatting class for YAML modules

SYNOPSIS


$self->die('YAML_PARSE_ERR_NO_ANCHOR', $alias);
$self->warn('YAML_LOAD_WARN_DUPLICATE_KEY');

DESCRIPTION

This module provides a "die" and a "warn" facility.
