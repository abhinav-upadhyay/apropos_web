NAME

fribidi_debug_status - FRIBIDI_BEGIN_DECLS should be used at the beginning of your declarations,

SYNOPSIS

#include <fribidi.h>
 
int fribidi_debug_status(void);

DESCRIPTION

So that C++ compilers don't mangle their names. Use FRIBIDI_END_DECLS at the end of C declarations.
