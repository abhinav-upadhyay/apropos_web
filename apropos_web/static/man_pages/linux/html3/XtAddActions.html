NAME

XtAddActions - register an action table

SYNTAX

void XtAddActions(XtActionList actions, Cardinal num_actions);

ARGUMENTS

actions
Specifies the action table to register.
num_args
Specifies the number of entries in this action table.

DESCRIPTION
