COMPIZ-DECORATOR(1) General Commands Manual COMPIZ-DECORATOR(1)

NAME

compiz-decorator - Automatic decorator loader

SYNOPSIS

compiz-decorator

DESCRIPTION

compiz-decorator is a program that automatically launches the best decorator based on your environment and what decorators are available.

AUTHOR
