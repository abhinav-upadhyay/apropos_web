NAME

dh_icons - Update Freedesktop icon caches

SYNOPSIS

dh_icons [debhelper options] [-n]

DESCRIPTION

dh_icons is a debhelper program that updates Freedesktop icon caches when needed, using the update-icon-caches program provided by GTK+2.12. Currently this program does not handle installation of the files, though it may do so at a later date. It takes care of adding maintainer script fragments to call update-icon-caches.

OPTIONS

-n, --noscripts
Do not modify maintainer scripts.
