NAME

bluetooth-agent - Bluetooth pass agent

SYNOPSIS

bluetooth-agent

DESCRIPTION

Bluetooth-agent is pass agent program for bluetooth.
 

USAGE

bluetooth-agent [--adapter adapter-path] [--path agent-path] <passkey> [<device>]
