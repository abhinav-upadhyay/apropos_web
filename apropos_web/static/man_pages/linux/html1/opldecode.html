rr_n {.nr rr_n 0 1 opldecode - Decode a Raster Object (opl) stream into human readable form. opldecode [options] <zjs-file opldecode decodes a Raster Object (opl) stream into human readable form.
A Raster Object stream is the printer langauge used by some Konica Minolta printers, such as the KM magicolor 2480 MF.
 
These are the options that can appear on the command line.
-d basename
Basename of .pbm file for saving decompressed planes.
-h
Print hex file offsets.
-o
Print file offsets.
-D level
Set Debug level [0].
 
Decode an Raster Object stream file created by foo2lava-wrapper -z1.
