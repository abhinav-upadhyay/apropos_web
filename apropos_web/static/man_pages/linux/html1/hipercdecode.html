rr_n {.nr rr_n 0 1 hipercdecode - Decode a HIPERC stream into human readable form. hipercdecode [options] <hiperc-file hipercdecode decodes a HIPERC stream into human readable form. Uncompressed and JBIG formats are handled.
An HIPERC stream is the printer langauge used by the Oki Data C310dn, C3100, C3200n, C3250n, C3300n, C3400n, C5100n, C5250n, C5500n, C5600, and the C5800n printers.
 
These are the options that can appear on the command line.
-d basename
Basename of .pbm file for saving decompressed planes.
-h
Print hex file offsets.
-o
Print file offsets.
-D level
Set Debug level [0].
 
Decode an HIPERC stream file created by foo2hiperc.
