rr_n {.nr rr_n 0 1 hbpldecode - Decode a HBPL stream into human readable form. hbpldecode [options] <hbpl-file hbpldecode decodes a HBPL stream into human readable form.
A HBPL stream is the printer langauge used by the Fuji-Xerox cp105b and the Dell 1250c printers.
 
These are the options that can appear on the command line.
-d basename
Basename of .pbm file for saving decompressed planes.
-h
Print hex file offsets.
-o
Print file offsets.
-D level
Set Debug level [0].
 
Decode an HBPL stream file.
