NAME

unity-2d-shell - Unity 2D Shell

SYNOPSIS

unity-2d-shell [-opengl] [-rootqml File.qml] displays some components of Unity 2D. It contains the launcher and dash. It functions as part Unity 2D, and is not meant to be used as a standalone application. It can however be used for testing or prototyping of only specific components of Unity 2D by using the rootqml argument.

OPTIONS

-opengl
Run the application using an openGL-enabled widget
qmlfile
The path of a QML file to load instead of the full shell
