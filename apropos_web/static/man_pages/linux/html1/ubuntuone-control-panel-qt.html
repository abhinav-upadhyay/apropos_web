NAME

ubuntuone-control-panel-qt - control panel for Ubuntu One

DESCRIPTION

usage: ubuntuone-control-panel-qt [-h] [--switch-to PANEL_NAME] [-a]
[--minimized] [--with-icon]

optional arguments:

-h, --help
show this help message and exit
--switch-to PANEL_NAME
Start Ubuntu One in the PANEL_NAME tab. Possible values are: folders, devices, settings, account
--minimized
Start Ubuntu One only in the notification area, with no visible window. Implies --with-icon
--with-icon
Start Ubuntu One with an icon in the notification area.
