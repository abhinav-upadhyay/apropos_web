NAME

squiggle - browser for SVG files
 

squiggle

DESCRIPTION

squiggle is a graphical browser for SVG files.
 

BUGS

squiggle is currently broken in debian. Startup will fail with a security error.
 

JAVA-WRAPPERS NOTE

This program is a shell script wrapper based on java-wrappers(7). You therefore benefit from several features; please see the java-wrappers(7) manual page for more information about them.
