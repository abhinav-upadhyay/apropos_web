NAME

sprof - Read and display shared object profiling data

SYNOPSIS

sprof -p⎪-c [-q]

DESCRIPTION

--call-pairs, -c

print list of count paths and their number of use

--flat-profile, -p

generate flat profile with counts and ticks

--graph, -q

generate call graph
