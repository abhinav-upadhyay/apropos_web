NAME

ocamlobjinfo - dump information about OCaml compiled objects

SYNOPSIS

ocamlobjinfo file ...

DESCRIPTION

Dump information contained in OCaml compiled objects. It works on .cmi, .cmo, .cma, .cmx, .cmxa, .cmxs files and pure bytecode executables.
 
ocamlobjinfo is able to show information regarding:
 
module names
 
unit names
 
declared primitives
 
imported interfaces
 
md5sums of imported interfaces
 
forced custom mode
 
extra C libraries needed
 
extra C flags needed
 
use of unsafe features
depending on its arguments.
