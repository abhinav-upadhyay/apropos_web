rr_n {.nr rr_n 0 1 xqxdecode - Decode a XQX stream into human readable form. xqxdecode [options] <xqx-file xqxdecode decodes a XQX stream into human readable form.
An XQX stream is the printer langauge used by some HP LaserJet printers, such as the HP LaserJet M1005 (MFP).
 
These are the options that can appear on the command line.
-d basename
Basename of .pbm file for saving decompressed planes.
-h
Print hex file offsets.
-o
Print file offsets.
-D level
Set Debug level [0].
 
Decode an XQX stream file created by foo2xqx.
