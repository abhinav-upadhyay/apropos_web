NAME

grub-bin2h - convert a binary file to a C header

SYNOPSIS

grub-bin2h [ OPTIONS] SYMBOL-NAME

DESCRIPTION

Convert a binary file to a C header.
-h, --help
display this message and exit
-V, --version
print version information and exit

REPORTING BUGS

Report bugs to <bug-grub@gnu.org>.

SEE ALSO

The full documentation for grub-bin2h is maintained as a Texinfo manual. If the info and grub-bin2h programs are properly installed at your site, the command
info grub-bin2h
should give you access to the complete manual.
