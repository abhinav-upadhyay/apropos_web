NAME

ocamldumpobj - disassembler for OCaml executable and .cmo object files

SYNOPSIS

ocamldumpobj file ...

DESCRIPTION

Disassembler for executable and .cmo object files compiled by OCaml.

SEE ALSO

ocamlc(1), ocamlopt(1), ocamlobjinfo(1).
