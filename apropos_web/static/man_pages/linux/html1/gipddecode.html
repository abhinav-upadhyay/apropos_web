rr_n {.nr rr_n 0 1 gipddecode - Decode a GIPD stream into human readable form. gipddecode [options] <gipd-file gipddecode decodes a Granite Image Printer Driver (GIPD) stream into human readable form. Granite Systems was acquired by Monotype Imaging.
A GIPD stream is the printer langauge used by the Lexmark X500 and the Dell 1125 MFP printers.
 
These are the options that can appear on the command line.
-d basename
Basename of .pbm file for saving decompressed planes.
-h
Print hex file offsets.
-o
Print file offsets.
-D level
Set Debug level [0].
 
Decode an GIPD stream file.
