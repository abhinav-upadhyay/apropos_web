NAME

md5pass - Create an MD5 password hash
 

SYNOPSIS

md5pass [PASSWORD] [SALT]
 

DESCRIPTION

md5pass creates an MD5 password hash. In the absence of a salt value on the command line, a random salt vector will be generated.
