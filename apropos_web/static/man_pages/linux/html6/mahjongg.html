NAME

Mahjongg - A matching game played with Mahjongg tiles

SYNOPSIS

mahjongg [OPTION...]

DESCRIPTION

GNOME Mahjongg, or Mahjongg for short, is a solitaire (one player) version of the classic Eastern tile game, Mahjongg.

OPTIONS

This program only accepts the standard GNOME and GTK options.

AUTHORS

mahjongg was written by Francisco Bustamante, Max Watson and many others.
