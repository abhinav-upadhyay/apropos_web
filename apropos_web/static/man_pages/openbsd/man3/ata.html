NAME

atan, atanf, atanl — arc tangent functions of one variable

SYNOPSIS

#include <math.h>
double
atan(double x);
float
atanf(float x);
long double
atanl(long double x);

DESCRIPTION

The atan() function computes the principal value of the arc tangent of x in the range [-pi/2, +pi/2]. The atanf() function is a single precision version of atan(). The atanl() function is an extended precision version of atan().
