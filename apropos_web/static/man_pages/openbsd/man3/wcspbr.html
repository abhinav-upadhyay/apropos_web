NAME

wcspbrk — locate multiple wide characters in a wide string

SYNOPSIS

#include <wchar.h>
wchar_t *
wcspbrk(const wchar_t *s, const wchar_t *charset);

DESCRIPTION

The wcspbrk() function locates in the wide string s the first occurrence of any wide character in the wide string charset and returns a pointer to this wide character. If no wide characters from charset occur anywhere in s, wcspbrk() returns NULL.

SEE ALSO

strpbrk(3), wcschr(3), wcscspn(3), wcsrchr(3), wcsspn(3), wcsstr(3), wcstok(3), wmemchr(3)

STANDARDS

The wcspbrk() function conforms to ISO/IEC 9899:1999 (“ISO C99”) and was first introduced in ISO/IEC 9899/AMD1:1995 (“ISO C90, Amendment 1”).

HISTORY

The wcspbrk() function was ported from NetBSD and first appeared in OpenBSD 3.8.
