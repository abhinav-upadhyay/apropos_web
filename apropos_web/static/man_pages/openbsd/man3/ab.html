NAME

abs — integer absolute value function

SYNOPSIS

#include <stdlib.h>
int
abs(int j);

DESCRIPTION

The abs() function computes the absolute value of the integer j.

RETURN VALUES

The abs() function returns the absolute value.

SEE ALSO

cabs(3), floor(3), hypot(3), imaxabs(3), labs(3)

STANDARDS

The abs() function conforms to ANSI X3.159-1989 (“ANSI C89”).

HISTORY

The abs() function first appeared in Version 6 AT&T UNIX.

BUGS

The absolute value of the most negative integer remains negative.
