NAME

pom — display the phase of the moon

SYNOPSIS

pom [[[[[cc]yy]mm]dd]HH]

DESCRIPTION

The pom utility displays the current phase of the moon. Useful for selecting software completion target dates and predicting managerial behavior.
[[[[cc]yy]mm]dd]HH
Display the phase of the moon for a given time. The format is similar to the canonical representation used by date(1).

SEE ALSO

date(1)

BUGS

Times must be within range of the UNIX Epoch.

ACKNOWLEDGEMENTS

This program is based on routines from Practical Astronomy with Your Calculator, Third Edition by Peter Duffett-Smith.
