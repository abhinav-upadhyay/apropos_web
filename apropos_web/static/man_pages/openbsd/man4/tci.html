NAME

tcic — Databook PCMCIA controller driver

SYNOPSIS

tcic0 at isa? port 0x240 iomem 0xd0000 iosiz 0x4000
pcmcia* at tcic?

DESCRIPTION

The tcic driver provides support for the Databook DB86082, DB86084, DB86184, and DB86072 PCMCIA controllers.

SEE ALSO

intro(4), isa(4), pcmcia(4)

HISTORY

The tcic driver appeared in NetBSD 1.4. OpenBSD support first appeared in OpenBSD 2.8.
