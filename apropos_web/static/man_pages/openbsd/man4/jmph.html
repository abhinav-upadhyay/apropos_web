NAME

jmphy — JMicron JMP202/JMP211 10/100/Gigabit Ethernet PHY

SYNOPSIS

jmphy* at mii?

DESCRIPTION

The jmphy driver supports the JMicron JMP202 10/100 and JMP211 10/100/Gigabit Ethernet PHYs.

SEE ALSO

ifmedia(4), intro(4), mii(4), ifconfig(8)

HISTORY

The jmphy device driver first appeared in OpenBSD 4.5.

AUTHORS

The jmphy driver was written by Pyun YongHyeon for FreeBSD and ported to OpenBSD by Jonathan Gray <jsg@openbsd.org>.
