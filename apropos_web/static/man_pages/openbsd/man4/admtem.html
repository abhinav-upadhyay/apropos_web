NAME

admtemp — Analog Devices ADM1021 temperature sensor

SYNOPSIS

admtemp* at iic?

DESCRIPTION

The admtemp driver provides support for the Analog Devices ADM1021, Analog Devices ADM1023, Analog Devices ADM1032, Genesys Logic GL523SM, Global Mixed-mode Technology G781, Maxim 1617, NXP SA56004x, and Xeon embedded temperature sensors. The device possesses internal and external temperature sensors. These values are made available through the sysctl(8) interface.
On i386 machines, this driver also supports the Xeon embedded I2C temperature probes. In this case, however, only one temperature value is provided.

SEE ALSO

iic(4), intro(4), sensorsd(8), sysctl(8)

HISTORY

The admtemp driver first appeared in OpenBSD 3.9.
