NAME

uplcom — Prolific PL-2303 USB serial adapter

SYNOPSIS

uplcom* at uhub?
ucom* at uplcom?

DESCRIPTION

The uplcom driver provides support for the Prolific PL-2303 USB-to-RS232 Bridge chip.
The device is accessed through the ucom(4) driver which makes it behave like a tty(4).
