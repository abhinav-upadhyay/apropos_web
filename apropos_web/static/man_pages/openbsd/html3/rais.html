NAME

raise — send a signal to the current thread

SYNOPSIS

#include <signal.h>
int
raise(int sig);

DESCRIPTION

The raise() function sends the signal sig to the current thread.

RETURN VALUES

Upon successful completion, the value 0 is returned; otherwise the value -1 is returned and the global variable errno is set to indicate the error.

ERRORS

raise() will fail and no signal will be sent if:
[EINVAL]
sig is not a valid signal number.

SEE ALSO

kill(2)

STANDARDS

The raise() function conforms to ANSI X3.159-1989 (“ANSI C89”) and IEEE Std 1003.1-2008 (“POSIX.1”).
