NAME

uptime — show how long system has been running

SYNOPSIS

uptime

DESCRIPTION

The uptime utility displays the current time, the length of time the system has been up, the number of users, and the load average of the system over the last 1, 5, and 15 minutes. This is the “heading” information from w(1).

SEE ALSO

w(1)

HISTORY

The uptime command appeared in 3.0BSD.
