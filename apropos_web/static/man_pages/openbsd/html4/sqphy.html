NAME

sqphy — Seeq 80220/80221, 80223, 80225 and 84220 10/100 Ethernet PHY

SYNOPSIS

sqphy* at mii?

DESCRIPTION

The sqphy driver supports the Seeq 80220/80221, 80223, 80225 and 84220 10/100 Ethernet PHYs. The 80223 is a 3.3 volt version of the 80221. The 80225 is a stripped down version of the 80223. These PHYs are found on a variety of high-performance Ethernet interfaces.
