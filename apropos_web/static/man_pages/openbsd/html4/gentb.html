NAME

gentbi — Driver for generic 1000BASE-X ten-bit interfaces

SYNOPSIS

gentbi* at mii?

DESCRIPTION

The gentbi driver supports generic ten-bit interfaces that implement the 1000BASE-X protocol, including 1000BASE-SX, 1000BASE-LX, and 1000BASE-CX.
