NAME

ex — Intel EtherExpress Pro/10 and Pro/10+ Ethernet device

SYNOPSIS

ex0 at isa? port 0x320 irq 5

DESCRIPTION

The ex interface provides access to a 10 Mb/s Ethernet network via the Intel i82595 Ethernet chips. This includes the following models:
Intel EtherExpress Pro/10 ISA
Intel EtherExpress Pro/10+ ISA
