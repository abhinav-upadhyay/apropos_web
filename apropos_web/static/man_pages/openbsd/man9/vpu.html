NAME

vput — decrement the reference count for a vnode and unlock it

SYNOPSIS

#include <sys/param.h>
#include <sys/vnode.h>
void
vput(struct vnode *vp);

DESCRIPTION

Decrement the v_usecount field of the vnode vp and unlock the vnode.
This operation is functionally equivalent to calling VOP_UNLOCK(9) followed by vrele(9).

SEE ALSO

vnode(9), VOP_UNLOCK(9), vref(9), vrele(9)
