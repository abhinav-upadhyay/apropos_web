NAME

vs — X68K built-in voice synthesizer driver

SYNOPSIS

vs0 at intio0 addr 0xe92000 dma 3 dmaintr 106
audio* at vs?

DESCRIPTION

vs drives x68k's built-in voice synthesizer. It is implemented as an audio(4) device.

SEE ALSO

audio(4), intio(4)
