NAME

ahb — Adaptec 1742 SCSI adapter driver

SYNOPSIS

ahb0 at eisa? slot ? irq ?
scsibus* at ahb?

DESCRIPTION

The ahb driver implements support for the following card:

Adaptec
AHA-1742 EISA SCSI adaptor
