LCD(4) Kernel Interfaces Manual (HPPA) LCD(4)

NAME

lcd — front panel LCD display

SYNOPSIS

lcd0 at mainbus0

DESCRIPTION

The lcd driver provides support for the front panel LCD display found on newer HP PA-RISC workstations and servers. This LCD can be made to show a heartbeat based on the load average by setting the sysctl(3) variable machdep.lcd_blink to a non-zero value.

SEE ALSO

sysctl(3), intro(4), sysctl(8)

HISTORY

The lcd driver first appeared in OpenBSD 4.2. It was ported to NetBSD 6.0 by Adam Hoka and Nick Hudson.
