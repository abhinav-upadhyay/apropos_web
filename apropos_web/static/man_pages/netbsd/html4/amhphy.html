NAME

amhphy — Driver for the AMD 79c901 10BASE-T PHY

SYNOPSIS

amhphy* at mii? phy ?

DESCRIPTION

The amhphy driver supports the 10BASE-T portion of the AMD 79c901 HomePNA/10BASE-T PHY.
