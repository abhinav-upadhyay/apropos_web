NAME

iwm — Intel Wireless and Centrino Wireless IEEE 802.11 wireless network driver

SYNOPSIS

iwm* at pci? dev ? function ?

DESCRIPTION

The iwm driver provides support for the following cards:
Intel Wireless 7260
The following cards could be supported with minor changes:
Intel Wireless 7265
Intel Wireless 3160

The Intel Wireless 7260 is a PCIe Half Mini Card network adapter that operates in the 2.4GHz and 5GHz spectra.

By default, the iwm driver configures the adapter for BSS operation (aka infrastructure mode). This mode requires the use of an access point.

For more information on configuring this device, see ifconfig(8).
