VIAC7TEMP(4) Kernel Interfaces Manual VIAC7TEMP(4)

NAME

viac7temp — VIA C7 temperature sensor

SYNOPSIS

viac7temp* at cpu?

DESCRIPTION

The viac7temp driver supports temperature sensors found in VIA C7 processors. The available information is available through the envsys(4) API and the envstat(8) command.

SEE ALSO

coretemp(4)
