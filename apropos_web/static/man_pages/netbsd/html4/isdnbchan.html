NAME

isdnbchan — ISDN Raw B-Channel access driver

SYNOPSIS

pseudo-device isdnbchan count

DESCRIPTION

The isdnbchan driver provides an interface to the raw untranslated B-channel.

SEE ALSO

isdnd.rc(5), isdnd(8)
