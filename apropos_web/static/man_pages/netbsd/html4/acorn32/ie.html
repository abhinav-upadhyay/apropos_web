NAME

ie — Ethernet driver for Ether1 podule Ethernet interfaces

SYNOPSIS

ie0 at podulebus?

DESCRIPTION

The ie interface provides access to a 10 Mb/s Ethernet network.

HARDWARE

The Ethernet cards supported by the ie interface are:

Acorn Ether1 podule
