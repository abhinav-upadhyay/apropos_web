NAME

tsc — DECchip 21272 Core Logic chipset

SYNOPSIS

tsc* at mainbus?

DESCRIPTION

The tsc driver provides support for the DECchip 21272 Core Logic chipset (Tsunami) found on the 264DP OEM Board.
