NAME

dwlpx — DEC DWLPA and DWLPB PCI adapter

SYNOPSIS

dwlpx* at kft?
pci* at dwlpx?

DESCRIPTION

The dwlpx driver provides support for DEC DWLPA and DWLPB PCI adapter found on the AlphaServer 8x00 Systems.
