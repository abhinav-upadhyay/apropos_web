NAME

tsp — DECchip 21272 Core Logic chipset PCI controller

SYNOPSIS

tsp* at tsc?
pci* at tsp?

DESCRIPTION

The tsp driver provides support for the PCI controller on the DECchip 21272 Core Logic chipset.
