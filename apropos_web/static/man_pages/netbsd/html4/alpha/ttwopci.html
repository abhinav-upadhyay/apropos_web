NAME

ttwopci — DEC T2 Gate Array PCI controller

SYNOPSIS

ttwopci* at ttwoga?
pci* at ttwopci?

DESCRIPTION

The ttwopci driver provides support for CBUS->PCI bridge on the DEC T2 Gate Array found in the AlphaServer 2100/2100A.
