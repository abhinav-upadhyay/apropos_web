NAME

tsciic — DECchip 21272 Core Logic chipset I2C controller

SYNOPSIS

tsciic* at tsc?
iic* at tsciic?

DESCRIPTION

The tsciic driver provides support for the I2C controller on the DECchip 21272 Core Logic chipset.
