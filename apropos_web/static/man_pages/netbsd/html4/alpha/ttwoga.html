NAME

ttwoga — DEC T2 Gate Array

SYNOPSIS

ttwoga* at mainbus?

DESCRIPTION

The ttwoga driver provides support for the Digital Equipment Corp. T2 Gate Array, the CBUS->PCI bridge found in the AlphaServer 2100/2100A.
