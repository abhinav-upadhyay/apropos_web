NAME

tcasic — TURBOchannel host bus support

SYNOPSIS

tcasic* at mainbus0
tc* at tcasic?

DESCRIPTION

The tcasic driver provides support for the TURBOchannel host bus on the baseboard of the DEC 3000/300 and 3000/500 systems.
