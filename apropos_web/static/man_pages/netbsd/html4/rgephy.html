NAME

rgephy — Realtek 8169S/8110S/8211 internal 10/100/1000 PHY driver

SYNOPSIS

rgephy* at mii? phy ?

DESCRIPTION

The rgephy driver supports the internal PHY found on Realtek 8169S/8110S/8211 Ethernet adapters.
