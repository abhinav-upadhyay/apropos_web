NAME

micphy — Micrel KSZ9021 10/100/1000 PHY driver

SYNOPSIS

micphy* at mii? phy ?

DESCRIPTION

The micphy driver currently supports only Micrel KSZ9021RN PHY. The driver has a fixup for evbarm/cpsw(4) which requires Gig-E PHYs to adjust RGMII signal timing.
