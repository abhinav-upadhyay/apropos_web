NAME

colorbars — display ANSI color bars

SYNOPSIS

colorbars

DESCRIPTION

The colorbars command displays ANSI color bars on a color capable terminal using curses(3).

SEE ALSO

curses(3)

HISTORY

colorbars appeared in NetBSD 7.0.
