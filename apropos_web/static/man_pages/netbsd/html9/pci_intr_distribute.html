NAME

pci_intr_distribute

SYNOPSIS

#include <dev/pci/pcivar.h>

int
pci_intr_distribute(void *ich, const kcpuset_t *newset, kcpuset_t *oldset);

DESCRIPTION

The pci_intr_distribute function exists to assign an interrupt to a CPU.

If a driver (or the other kernel component) wishes to assign an interrupt to a CPU, it should pass the return value of pci_intr_establish() as ich argument, and it should pass the kcpuset to which it should be assigned as newset. To get the previous value, pass a non-NULL value to oldset.
