DESCRIPTION

The krb5_rcache structure holds a storage element that is used for data manipulation. The structure contains no public accessible elements.

krb5_rc_initialize() Creates the reply cache id and sets it lifespan to auth_lifespan. If the cache already exists, the content is destroyed.
