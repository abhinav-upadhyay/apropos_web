CPROJ(3) Library Functions Manual CPROJ(3)

NAME

cproj, cprojf, cprojl — complex projection functions

SYNOPSIS

#include <complex.h>

double
cproj(double complex z);

float
cprojf(float complex z);

long double
cprojl(long double complex z);

DESCRIPTION

These functions compute a projection of z onto the Riemann sphere: z projects to z, except that all complex infinities (even those with one infinite part and one NaN part) project to positive infinity on the real axis. If z has an infinite part, then cproj() shall be equivalent to:

INFINITY + I * copysign(0.0, cimag(z))

RETURN VALUES

These functions return the value of the projection onto the Riemann sphere.

ERRORS

No errors are defined.
