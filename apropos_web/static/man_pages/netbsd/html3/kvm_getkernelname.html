KVM_GETKERNELNAME(3) Library Functions Manual KVM_GETKERNELNAME(3)

NAME

kvm_getkernelname — get kernel name of opened kvm descriptor

LIBRARY

Kernel Data Access Library (libkvm, -lkvm)

SYNOPSIS

#include <kvm.h>

const char *
kvm_getkernelname(kvm_t *kd);

DESCRIPTION

This function returns a string containing the kernel name used from the kvm descriptor obtained by a previous kvm_open(3) or kvm_openfiles(3) call.
