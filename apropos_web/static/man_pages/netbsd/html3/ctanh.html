NAME

ctanh, ctanhf, ctanhl — complex hyperbolic tangent functions

SYNOPSIS

#include <complex.h>

double complex
ctanh(double complex z);

float complex
ctanhf(float complex z);

long double complex
ctanhl(long double complex z);

DESCRIPTION

These functions compute the complex hyperbolic tangent of z.

RETURN VALUES

These functions return the complex hyperbolic tangent value.

ERRORS

No errors are defined.

catanh(3), IEEE Std 1003.1-2001 (“POSIX.1”) <complex.h>
