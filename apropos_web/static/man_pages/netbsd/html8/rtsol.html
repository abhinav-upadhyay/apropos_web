NAME

rtsold — router solicitation daemon

DESCRIPTION

rtsold and rtsol have been removed from NetBSD as their functionality is now included in dhcpcd(8).

For example to send a routing solicitation message and print information about it without changing the actual routing table:

	dhcpcd -dB6T --nodhcp6 <interface>
