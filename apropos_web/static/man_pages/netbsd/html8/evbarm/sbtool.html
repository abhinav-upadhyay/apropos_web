NAME

sbtool — manipulate sb files

SYNOPSIS

sbtool [-?, --help] [-b, --binary] [-d, --debug] [-k, --key file] [-q, --quiet] [-V, --verbose] [-v, --version] [-x, --extract -value] [-z, --zero-key] sb-file

DESCRIPTION

The sbtool programm is used to manipulate sb boot files for Freescale i.MX23 and i.MX28 application processors.

SEE ALSO

elftosb(8), sbkeygen(8)
