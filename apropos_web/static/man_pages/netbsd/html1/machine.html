NAME

machine — print machine type

SYNOPSIS

machine

DESCRIPTION

The machine command displays the machine type. It is equivalent to ‘uname -m’.

SEE ALSO

make(1), uname(1)

HISTORY

A machine command appeared in 4.3BSD-Reno.
