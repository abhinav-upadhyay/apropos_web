NAME

intro — introduction to the Lua kernel bindings

DESCRIPTION

This section provides an overview of the Lua kernel bindings, their functions, error returns and other common definitions and concepts.

EXAMPLES

An example module is available in the src/sys/modules/examples/ directory hierarchy.

SEE ALSO

lua(1), luac(1), intro(3lua), lua(4), pmf(9lua), systm(9lua)

HISTORY

An intro manual for Lua kernel bindings appeared in NetBSD 7.0.
