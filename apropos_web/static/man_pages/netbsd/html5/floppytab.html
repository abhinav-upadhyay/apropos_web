NAME

floppytab — parameters for floppy diskettes

DESCRIPTION

The floppytab file defines parameters for “floppy” diskettes. The format of the file matches the arguments passed to the fdformat(1) utility. When invoked with the -t flag, fdformat(1) can use the known floppy formats from /etc/floppytab directly.

FILES

/etc/floppytab

SEE ALSO

eject(1), fdformat(1)
